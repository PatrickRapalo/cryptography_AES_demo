Advanced Encryption Standard (AES-256)
AES uses symmetric key cryptography to transform plaintext into ciphertext through 14 rounds of mathematical operations. Your password is expanded into 15 unique round keys, with each round using a different key for maximum security. The same secret key encrypts and decrypts the data.

RSA (Rivest-Shamir-Adleman) - Asymmetric Encryption
RSA uses two different keys: a public key for encryption (which anyone can know) and a private key for decryption (which only you keep secret). This solves the key distribution problem of symmetric encryption.
üîì
Public Key
Shared openly. Anyone can encrypt messages to you using your public key.
üîê
Private Key
Kept secret. Only you can decrypt messages encrypted with your public key.
üßÆ
Math Foundation
Based on prime factorization. Easy to multiply primes, extremely hard to factor the result.

Trade-off
More secure key exchange, but slower than symmetric encryption (1000x slower!).
